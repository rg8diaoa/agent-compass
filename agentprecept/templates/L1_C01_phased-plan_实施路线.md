## 使用说明

本文件是实施路线的**通用模板**。新项目初始化时自动复制到 `docs/`，由 Agent 和人类协作填充各阶段的具体内容。

### Agent 自动行为

- **会话首次启动**：读取本文件，检查当前阶段
- **完成模块后**：更新对应行的验收状态
- **阶段门禁通过后**：建议推进到下一阶段
- **每 5 轮对话**：对照本文件检查进度偏差

### 模式感知

如果项目使用 agentprecept 的多模式系统（lite / normal / pro / max），本文件的详细程度应与当前模式匹配：
- **lite**：标题 + 3 行要点，标注假设
- **normal**：完整模块列表 + 验收标准
- **pro**：额外标注每阶段的审计门禁
- **max**：关联 agentprecept serve 的自动化阶段转移
