## §1 文档命名格式

```
L{Level}_{Category}{NN}_{Slug}_{Title}.md
```

| 字段 | 格式 | 说明 |
|------|------|------|
| `L{Level}` | L1 / L2 / L3 / L4 | 阅读层级 |
| `{Category}` | 单字母 A-P | 分类码 |
| `{NN}` | 两位数字 01-99 | 分类内序号，连续 |
| `{Slug}` | kebab-case，≤25 字符 | 英文短标识 |
| `{Title}` | ≤8 字 | 精简中文标题 |
